## 📖 **О проекте**

**VideoPlatform** — это современная веб-платформа для просмотра, загрузки и управления видеоконтентом. Проект выполнен в рамках расчётно-графической работы по дисциплине «Информационные технологии».

### ✨ **Основные возможности**

| Функция | Описание |
|---------|----------|
| 🔐 **Аутентификация** | Регистрация и вход с использованием JWT-токенов |
| 📤 **Загрузка видео** | Поддержка форматов MP4, MOV, AVI |
| 🎥 **Потоковое видео** | Воспроизведение с возможностью перемотки |
| ❤️ **Лайки** | Оценка понравившихся видео |
| 💬 **Комментарии** | Обсуждение видео под каждым роликом |
| 👁️ **Просмотры** | Автоматический счётчик просмотров |
| 🎨 **Современный дизайн** | Киберпанк-стиль с анимациями |
